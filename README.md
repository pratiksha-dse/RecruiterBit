# RecruitBit

# About the Project
Recruitment Management Tool

# Tech Stack - MERN
- Front-end - ReactJS
- Database - MongoDB Atlas
- Back-end - NodeJS (ExpressJS)
- Google API used for GSignIn

# Installing Packages

    > npm install
    > cd client
    > npm install
    
    Also add a config.env file in the root directory with the format:
    MONGODB = "<MONGODB_URL>"

# Development Mode
    - Terminal 1 (Back-End)
    > npm start
    - Terminal 2 (Front-End)
    > cd client
    > npm start
